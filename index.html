📖 기능 매뉴얼
📅 당직표 생성
설정 탭 → 기준월 선택 → 공휴일 입력 → 당직표 생성 클릭
• 전체 이력 기반 편차 자동 계산 (누적 횟수 적은 사람 우선)
• 데스크 전담 직원은 격일로 8슬롯 전담 배정
• 공휴일·주말 자동 제외
🔄 교체 기능
교체 모드 버튼 → A슬롯 교체 클릭 → B슬롯 교체 클릭 → 교체 확정
• 일반직원 ↔ 일반직원만 교체 가능
• 데스크 전담 ↔ 데스크 전담만 교체 가능
• 혼용 교체 시 경고 메시지 표시
✏️ 슬롯 수동 변경 (관리자 전용)
관리자 모드 잠금 해제 → 각 슬롯 옆 변경 버튼 → 직원 명부에서 선택
• 전체 직원 명부에서 이름 검색 후 선택
• 선택 즉시 당직표 반영 + 이력 자동 기록
🏖 데스크 전담 휴무 지정 (관리자 전용)
설정 탭 → 데스크 전담 휴무 지정 → 날짜 선택 → 적용
• 해당일 자동으로 일반직원 8명 편차 기준 배정 (대타 출근)
• 배정 후 슬롯별 수동 변경 가능
• 중요: 격일제 직원의 그날 근무는 본인 몫(연차)으로 유지됨
  → 정당하게 쓴 연차로 다음 달 누적 정산에서 불이익 없음
👤 직원 관리 (관리자 전용)
직원 관리 탭 → 직원 추가 버튼
• 일반직원: 이름/직급/부서 입력
• 데스크 전담(격일제): 격일제 근무 체크 → 시작일 입력
• 격일제 직원 2명이면 시작일 하루 차이로 설정 → 자동 핑퐁 배정
• 시작 순번 우선순위:
  ① 설정 탭의 첫 시작자 지정 (있으면 절대 우선)
  ② 이전 달 누적이 적은 사람
  ③ 누적이 같으면 신규(나중) 합류자 우선
📊 배정 통계
통계 탭에서 이번달 배정 현황 확인
• 전체 누적 횟수 기준 막대그래프
• 데스크 전담 직원은 별도 표시 (편차 계산 제외)
📅 캘린더 뷰 (NEW)
당직표 화면 상단의 📅 캘린더 버튼으로 달력 형태 전환
• 격일제(데스크 전담) 직원은 A·B 라벨로 한눈에 표시
• 월 하단에 A·B 근무일수 / 차이 / 일반 배정일 / 총 슬롯 요약
• 주말·공휴일·휴무대체 색상 구분
🔁 격일제 누적 정산 (NEW)
격일제 직원이 2명 등록되면 자동 핑퐁(A→B→A…) 배정
• 매월 시작 순번은 이전 달까지의 누적 근무일수가 적은 사람이 먼저
• 예) 지난달 A 10회 / B 11회 → 이번달은 A부터 시작 → 이후 핑퐁
• 시작일 이전인 직원은 배정 제외, 이후 자동 합류
↺ 데스크 복구 (NEW · 관리자 전용)
설정 탭의 "데스크 전담 휴무 지정"으로 일반직원 8명 배정된 날
→ 당직표 화면 해당 날짜 상단 ↺ 데스크 복구 버튼 클릭
• 원래 들어갔어야 할 격일제 직원으로 복귀
• 일반직원 휴무대체 이력은 자동 제거되고 격일제 배정 이력으로 교체

• 평일마다 다른 격일제 직원과 번갈아 근무하며, 주말·공휴일은 휴무

1. 동료와 일정 교체 — 다른 격일제 직원과 근무일을 맞바꿔 쉴 수 있어요.

2. 연차 사용 — 그날 일반직원이 대타로 배정됩니다.
   본인 근무일수는 그대로 유지되어 다음 달 누적 정산에 영향 없어요.
👥 일반직원이라면
• 캘린더/리스트에서 본인 이름이 있는 날·시간대가 당직 차례
• 자동 배정 규칙: 3일 이상 간격으로 떨어뜨리고, 전체 직원 간 편차 1~2회 이내로 균등 배정
• 부득이하게 변경 필요한 경우 → 관리자에게 요청 (교체 또는 슬롯 변경)
• 데스크 전담 직원 연차일에는 일반직원 8명이 대타로 배정되기도 합니다
🖨 인쇄 / 공유 / 엑셀
• 화면 우측 상단의 인쇄 버튼: 깔끔하게 출력

• 표 복사: 클립보드에 복사 → 엑셀이나 메신저에 붙여넣기 가능

❓ 자주 묻는 질문
Q. 동료와 협의해서 교체하고 싶어요
위의 🔄 교체 기능으로 직접 교체 가능합니다. 동료와 사전 협의만 끝내면 본인이 바로 진행하면 돼요.
Q. 휴무대체로 들어왔는데 그날 시간이 안 됩니다
부득이하게 교체가 필요하면 동료와 협의 후, 협의된 사람의 이름을 관리자에게 알려주세요. 관리자가 교체 처리해드립니다.
Q. 격일제 직원인데 그날 쉬어야 합니다
두 가지 방법이 있어요. ①동료와 일정 교체(위 🔄 교체 기능으로 직접 진행) ②연차 사용(일반직원이 대타로 배정됨). 어느 쪽이든 본인 근무일수는 유지되어 다음 달 시작 순번에 영향 없습니다.
Q. 어제·지난주 당직표는 어디서 보나요?
당직표 페이지 상단의 전달 버튼(예: 5월)을 누르면 지난달 기록을 미리보기로 볼 수 있어요. 편집은 불가하지만 확인은 가능합니다.
