📖 기능 매뉴얼
📅 당직표 생성
설정 탭 → 기준월 선택 → 공휴일 입력 → 당직표 생성 클릭
• 전체 이력 기반 편차 자동 계산 (누적 횟수 적은 사람 우선)
• 데스크 전담 직원은 격일로 8슬롯 전담 배정
• 공휴일·주말 자동 제외
🔄 교체 기능
교체 모드 버튼 → A슬롯 교체 클릭 → B슬롯 교체 클릭 → 교체 확정
• 일반직원 ↔ 일반직원만 교체 가능
• 데스크 전담 ↔ 데스크 전담만 교체 가능
• 혼용 교체 시 경고 메시지 표시
✏️ 슬롯 수동 변경 (관리자 전용)
관리자 모드 잠금 해제 → 각 슬롯 옆 변경 버튼 → 직원 명부에서 선택
• 전체 직원 명부에서 이름 검색 후 선택
• 선택 즉시 당직표 반영 + 이력 자동 기록
🏖 데스크 전담 휴무 지정 (관리자 전용)
설정 탭 → 데스크 전담 휴무 지정 → 날짜 선택 → 적용
• 해당일 자동으로 일반직원 8명 편차 기준 배정
• 배정 후 슬롯별 수동 변경 가능
👤 직원 관리 (관리자 전용)
직원 관리 탭 → 직원 추가 버튼
• 일반직원: 이름/직급/부서 입력
• 데스크 전담(격일제): 격일제 근무 체크 → 시작일 입력
• 격일제 직원 2명이면 시작일 하루 차이로 설정 → 자동 핑퐁 배정
📊 배정 통계
통계 탭에서 이번달 배정 현황 확인
• 전체 누적 횟수 기준 막대그래프
• 데스크 전담 직원은 별도 표시 (편차 계산 제외)
