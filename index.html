v1.0 · 2026-05
데스크 당직표 자동생성기 최초 배포
• 기준월 + 공휴일 설정 → 평일 자동 추출
• 하루 8슬롯 × 서로 다른 8명 배정
• 근무일 기준 최소 3일 간격 적용 (토·일 미포함)
• 누적 이력 기반 균등 분배 (편차 1~2회 이내)
• 당직 교체 기능 (교체 모드 → A/B 선택 → 확정)
• 시간대 고정 배정 (직원관리에서 설정)
• 💾 저장 버튼 — 명시적 저장 시에만 반영
• 📥 엑셀 다운로드 (신관 2층 입구 당직표 양식)
• 배정 이력 누적 저장 / 월별 삭제
• 관리자 비밀번호 잠금 (설정·통계·이력·직원관리·매뉴얼)
• 방문대장 작성 안내 (AEO 필수 기재 항목 포함)
• Google Apps Script 연동 — 클라우드 저장/불러오기
• GitHub Pages 배포 지원
📖 기능 매뉴얼
📅 당직표 생성
1. 설정 탭 → 기준월 선택 (예: 2026-06)
2. 공휴일 자동 입력 버튼 클릭 또는 날짜 직접 추가
3. 당직표 생성 → 버튼 클릭
4. 생성된 당직표는 당직표 탭에서 확인
※ 매달 생성하면 이전 달 이력이 자동 반영되어 연속 공평 배분
⇄ 당직 교체
1. 당직표 탭 상단 ⇄ 교체 모드 버튼 클릭
2. 각 슬롯에 교체 버튼 표시됨
3. 교체할 A 슬롯의 교체 버튼 클릭 → B 슬롯 교체 버튼 클릭
4. 상단 패널에서 두 슬롯 확인 후 교체 확정 클릭
5. 당직표 즉시 반영 + 이력에 교체 기록 자동 저장
※ 교체된 슬롯에는 주황색 '교체' 배지 표시
📌 시간대 고정
1. 직원 관리 탭 → 해당 직원 편집 클릭
2. 고정 시간대 드롭다운에서 슬롯 선택 (슬롯 1~8)
3. 저장 후 → 이후 모든 달 당직표에서 해당 슬롯만 배정됨
※ 고정된 직원은 당직표에 주황색 '고정' 배지 표시
🔒 관리자 모드
• 1회 인증 후 관리자 모드 유지 (페이지 새로고침 전까지)
• 사이드바 하단 🔓 관리자 잠금 버튼으로 즉시 재잠금
• 비밀번호 변경: 설정 탭 → 비밀번호 변경 섹션
🗂️ 이력 관리
• 당직표 생성/교체 시 자동 저장 (브라우저 종료 후에도 유지)
• 이력 탭에서 월별로 확인 및 삭제 가능
• 이력이 쌓일수록 배정 균등도 향상 (누적 기반 공평 분배)
• 표 복사 버튼: 날짜·요일·시간·성명·직급·부서 형식으로 클립보드 복사 → 엑셀 붙여넣기 가능
⚙️ 배정 규칙
• 하루 8개 시간대 → 서로 다른 8명 배정 (중복 없음)
• 근무일 기준 최소 3일 간격 (토·일 미포함)
  예) 월요일 당직 → 다음 가능일 목요일
  예) 금요일 당직 → 다음 가능일 수요일
• 전체 배정 횟수 편차 1~2회 이내 균등 분배
• 인원이 부족해 3일 간격 불가 시 간격 조건 자동 완화
